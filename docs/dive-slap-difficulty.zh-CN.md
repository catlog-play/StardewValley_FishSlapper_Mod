# 跳水扇鱼难度说明

当前版本的跳水扇鱼不再使用固定 `5 次 / 5 秒`。

## 基础档位

基础难度使用原版钓鱼小游戏里的 `BobberBar.difficulty`：

| 条件 | 目标次数 | 时限 |
| --- | ---: | ---: |
| `difficulty < 40` | 5 hits | 3.0s |
| `40 <= difficulty < 70` | 6 hits | 2.7s |
| `70 <= difficulty < 90` | 7 hits | 2.5s |
| `90 <= difficulty < 105` | 8 hits | 2.2s |
| `difficulty >= 105` 或 `bossFish` | 9 hits | 2.0s |

## 行为修正

随后再读取 `Data/Fish` 里的鱼行为参数做修正：

| behavior | 修正 |
| --- | --- |
| `smooth` | 无修正 |
| `floater` | 无修正 |
| `mixed` | `+1 hit`，`-0.2s` |
| `sinker` | `+1 hit` |
| `dart` | `+1 hit`，`-0.2s` |

## 说明

- 每次跳水扇鱼会在回到岸上时固定结算 `20` 血量和 `50` 体力。
- 无论本次跳水最终成功还是失败，都会在回岸时结算这笔代价。
- 如果当前数值不够扣，或扣完后会变成 `0`，则在回岸时直接走原版血量归零 / 体力耗尽流程。
- 传奇鱼会直接走最高基础档位。
- 跳水成功固定取消 perfect 加成，不再提供配置开关。
